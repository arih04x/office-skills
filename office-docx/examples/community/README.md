# Community Cases: office-docx

Use this area for public Word/DOCX cases.

- `good/`: reproducible high-quality DOCX workflows.
- `corrections/`: bad output, diagnosis, fix, and better result.
- `anti-patterns/`: workflows to avoid and the preferred direction.

Keep one-off outputs and previews in `examples/`; keep reusable templates and
starter code in `assets/`.
