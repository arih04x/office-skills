# Soft Roadmap 风格

该风格来自一类中文科研路线图：白底、横向多栏、顶部阶段标题、虚线分区框、浅色任务卡片、蓝色核心节点和黑色依赖箭头。适合开题报告、毕业论文技术路线、科研项目路线、任务分解、研究内容与产出说明。

## 模板文件

模板位于：

```text
assets/drawio-templates/
```

可用模板：

| 文件 | 用途 |
| --- | --- |
| `soft-roadmap-3col.drawio` | 问题目标 -> 方法框架 -> 验证产出 |
| `soft-research-roadmap-4col.drawio` | 项目愿景 -> 核心挑战 -> 实施路径 -> 阶段成果 |
| `soft-roadmap-5col.drawio` | 背景问题 -> 关键难点 -> 研究任务 -> 验证实验 -> 成果产出 |

模板本身不依赖外部图片，所有元素都是 draw.io 原生可编辑对象。

模板预览图放在：

```text
examples/showcase/previews/soft-research-roadmap-4col.preview.png
```

## 视觉规则

- 背景保持白色。
- 每一列使用黑色虚线容器。
- 顶部阶段标题使用圆角矩形，描边比内部卡片略粗。
- 内部任务卡片使用浅黄、浅红、浅绿等低饱和填充。
- 中心关键节点使用蓝底白字。
- 箭头使用黑色，线宽约 2，优先表达跨列依赖和列内闭环。
- 字体用中文无衬线，字号稳定，不使用大段文字。
- 图内不放大标题，标题交给 Word/PPT/论文图题。

## 4 栏模板基准参数

`soft-research-roadmap-4col.drawio` 已按参考图压缩为接近原图的横向比例：

- 画布：约 `875 x 436`。
- 标题框：约 `176 x 43`，第三列略宽。
- 内容容器：约 `175 x 326`。
- 第 1/4 栏核心卡片：约 `153 x 66-68`。
- 第 2 栏难题卡片：约 `142 x 32`。
- 第 3 栏任务卡片：约 `150 x 44`。
- 列间距保持紧凑，不做宽松大画布。
- 普通卡片不加粗，蓝色核心节点加粗。

## 结构规则

### 3 栏

用于内容较少的图：

```text
问题目标 -> 方法框架 -> 验证产出
```

每栏 2-4 个卡片即可。中间栏保留 3-4 个任务卡片。

### 4 栏

用于最常见的科研路线图：

```text
项目愿景 -> 核心挑战 -> 实施路径 -> 阶段成果
```

推荐结构：

- 第 1 栏：上/中/下三块，中间蓝色核心目标。
- 第 2 栏：上半部分黄色挑战，下半部分红色挑战。
- 第 3 栏：按步骤编号纵向排列，黄色步骤在上，红色步骤在下。
- 第 4 栏：上/中/下三块，中间蓝色阶段成果或闭环。

### 5 栏

用于开题、基金、项目书：

```text
背景问题 -> 关键难点 -> 研究任务 -> 验证实验 -> 成果产出
```

适合把“验证实验”单独作为一列，避免研究任务和实验混在一起。

## 改写步骤

1. 复制最接近的 `.drawio` 模板到任务输出目录。
2. 保留列宽、虚线容器、标题高度、卡片圆角和蓝色核心节点。
3. 替换栏目标题和卡片文字。
4. 根据内容增删卡片，但每列保持垂直居中和统一间距。
5. 调整箭头，只保留关键依赖，不把所有节点都连满。
6. 如果要插入 Word/PPT，导出 SVG；如果包含位图层，导出 PNG。

## 与生成图片结合

该风格本身以清晰结构为主，通常不需要生成图片。如果用户要求更强视觉风格：

- 可以生成一张低对比背景纹理或侧边装饰图。
- 生成图片必须放在底层。
- 所有卡片、文字、箭头仍然使用 draw.io 原生元素。
- 背景透明度要低，不能影响黑色箭头和中文标签可读性。

## 质量检查

交付前检查：

- 四周留白均衡。
- 标题框宽度一致。
- 虚线容器高度一致。
- 同列卡片左右对齐。
- 跨列箭头不穿过文字。
- 蓝色核心节点在视觉上确实是中心。
- 文本没有超过卡片边界。
- `.drawio` 源文件保留，导出图仅作为插入资产。
