# Example: Hybrid Office Figure

User request:

> Create a research framework figure for Word using the onij style. It should
> have a generated visual layer, but the labels and arrows must remain editable.

Implementation:

1. Use `assets/styles/onij.png` as the style reference.
2. Generate a background or visual panel with `assets/node-image-starter/`.
3. Create a `.drawio` figure that places the generated image as a layer.
4. Add editable labels, arrows, modules, and callouts in draw.io.
5. Export PNG for Word insertion and keep `.drawio` as the editable source.

Reference files:

- `references/image-generation.md`
- `references/drawio-figures.md`
- `references/office-integration.md`
- `references/validation.md`
