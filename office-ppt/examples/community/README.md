# Community Cases: office-ppt

Use this area for public PowerPoint/PPTX cases.

- `good/`: reproducible high-quality slide or deck workflows.
- `corrections/`: bad output, diagnosis, fix, and better result.
- `anti-patterns/`: workflows to avoid and the preferred direction.

Keep one-off outputs and previews in `examples/`; keep reusable templates,
styles, and starter code in `assets/`.
