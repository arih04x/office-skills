# Sticker GIFs

Use this reference for animated stickers, emoji packs, small expression GIFs,
and branded mascot motion.

## Default Sticker Pack

| Action | Filename | Caption |
| --- | --- | --- |
| Happy waving | `hi` | `Hi~` / `嗨~` |
| Laughing | `laugh` | `LOL` / `哈哈哈` |
| Crying | `cry` | `Boo-hoo` / `呜呜呜` |
| Heart gesture | `love` | `Love ya` / `爱你哦` |

## Rules

- 1:1 canvas.
- Subject centered.
- White or transparent-looking simple background.
- Caption at bottom, large, high contrast.
- Loop smoothly in 1-2 seconds.
- Keep caption text stable; do not animate it with warping effects.

## Workflow

1. Create or select one source image per sticker.
2. Add stable local caption.
3. Add simple loop motion: bounce, pulse, tilt, wave proxy, sparkle, or zoom.
4. Export GIF and inspect frame count/size.
