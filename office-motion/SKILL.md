# Office Motion

Use this skill when the user needs a promotional GIF, animated visual asset, PPT
motion insert, short looping animation, or GIF/video conversion for Office and
marketing workflows.

Default to a local, repeatable pipeline first: Pillow for generated GIFs and
image sequences, FFmpeg for MP4/video conversion when available, and optional
image generation only for source artwork. Keep exact text stable and readable.

## Operating Model

- Use `scripts/motion_toolkit.py` for deterministic GIF creation, conversion,
  and inspection.
- Treat generated image/video APIs as optional upstream sources, not required
  runtime dependencies.
- Keep final text overlays in local frames when exact wording matters.
- Deliver PPT-safe assets: `.gif` for simple loops, `.mp4` for smoother or
  longer motion, and a still PNG fallback when appropriate.
- Use 16:9 for PPT banners, 1:1 for stickers, 9:16 for stories/reels, and
  social-specific dimensions when requested.

## Implementation Steps

1. Identify target surface: PPT, WeChat, web, social post, email, or sticker.
2. Choose dimensions and format: GIF for compatibility, MP4 for smooth motion.
3. Create or gather source artwork.
4. Keep captions and CTA text stable in local rendering.
5. Generate the motion asset:

   ```bash
   python scripts/motion_toolkit.py promo-gif --title "Launch Update" --subtitle "Three workflow wins" --cta "Read more" --out promo.gif
   ```

6. Inspect output:

   ```bash
   python scripts/motion_toolkit.py inspect --input promo.gif
   ```

7. For PPT, insert the GIF/MP4 and verify playback in PowerPoint when possible.

## Design Rules

- Loop duration: 2-5 seconds for promotional GIFs; 1-2 seconds for stickers.
- Text should not warp, rotate, shimmer, or be generated by a video model.
- Use a still frame fallback for PowerPoint thumbnails and static exports.
- Keep file size practical: under 8 MB when possible, under 3 MB for chat/email.
- Avoid dense paragraphs; animated assets should carry one message.
- Use MP4 instead of GIF for photorealistic, long, or gradient-heavy motion.
